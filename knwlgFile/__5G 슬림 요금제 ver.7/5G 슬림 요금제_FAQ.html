  FAQ
 

Q1. 데이터투게더/테더링/데이터쉐어링 데이터 사용량은 어떻게 알 수 있나요?
A1. KOS/마이의 사용량 조회 페이지에서 확인 가능하시며, 사용량 소진 시에 실시간 사용량 안내 문자로 안내됩니다.

Q2. 5G 슬림 QoS형 요금제는 데이터 밀당 서비스가 제공되나요?
A2. 아닙니다. 5G 슬림 QoS형 요금제는 QoS 무제한 요금제로 이월과 당겨쓰기 서비스를 제공하지 않습니다.

 

[24.01.18 버전업 내용4] Q3~Q8번 추가

Q3. 이월/종량 요금제의 기본 데이터 소진 후 공유 데이터 사용 시 초과 요금이 발생하나요?

A3. 아니요, 200Kbps로 속도 제어되며, 초과요금은 발생하지 않습니다. (QoS형 요금제와 동일)

 

Q4. 이월/종량 요금제와 LTE 밀당 요금제(LTE 베이직 등)간 변경 시 이월 데이터는 승계되나요?

A4. 아니요, 5G 이월/종량 요금제 5종간 변경을 제외하고 승계되지 않습니다. 

 

Q5. 이월/종량 요금제 이용 중 발생한 초과요금도 선택약정 할인이 적용되나요?

A5. 아니요, 적용되지 않습니다. 

 

Q6. 이월/종량 요금제 이용 중 초과요금 발생 여부 및 초과금액은 고객이 어떻게 알 수 있나요?

A6. 기본 제공 데이터 50%/80%/100% 소진 시점과 초과금액이 1/2/3/4/6/8/10만원 발생 시 안내문자가 발송됩니다. 

     또한 kt.com 내 마이 또는 마이케이티 앱에서도 고객이 직접 확인 가능합니다. 

 

Q7. 이월/종량 요금제의 종량 과금 3구간에서 속도제어 해제('5G 데이터 초과 요금 상한' 부가서비스 해지)는 KOS에서만 가능한가요?

A7. 아니요, kt.com 내 마이 또는 마이케이티 앱에서 고객이 직접 가입/해지 가능합니다. 

 

Q8. 동일 월정액의 QoS형 요금제와 이월/종량 요금제간 변경 시 월중 요금제 변경 횟수에 포함되나요? 

A8. 네, QoS 요금제와 이월/종량 요금제는 별도 요금제로 변경 횟수에 포함됩니다.
