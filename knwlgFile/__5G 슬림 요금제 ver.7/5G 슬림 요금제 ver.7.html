Carryover data

ㅇ The remaining data from the current month will be automatically carried over (no separate application required from the customer) and can be used until the end of the following month .

 ex) The remaining data from January will be carried over to February and can be used until the end of February.

ㅇ Carried-over basic data can be used as shared data, but carried-over Y-dump data cannot be used as shared data.

ㅇ Data deduction order: Data taken from Family Box/Y Box → Long-term customer coupon → Y discount data carried over → Basic data carried over → Current month’s Y discount data →

                             Current month's basic data → Data recharge

 - The remaining data for the current month (Y-dump data + basic data) is automatically carried over to the next month and then deducted first.

 ※ However, if a Y-Dum user turns 35 and terminates the subscription on the 1st of the following month, the remaining Y-Dum additional data from the previous month cannot be carried over.
